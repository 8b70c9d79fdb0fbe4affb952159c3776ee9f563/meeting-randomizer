# Meeting Randomizer #

Chooses randomly a weighted language of the day and today's order of
parcipitants.

## Usage

Copy `data_TEMPLATE.py` to `data.py` and fill in your langages and people.

Then:

```
$ ./meeting-randomizer.py 
Language of the day:
====================
German

Order of the day:
=================
Joe
Bob
Alice
Melinda
```
